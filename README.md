# 🌤️ SkySync Weather: Real-Time Insights

**A modern, responsive web application focusing on efficient RESTful API integration and dynamic UI/UX management.**

This weather application provides real-time meteorological data with a sleek interface. It was developed to demonstrate seamless data fetching from external services and conditional rendering based on live environmental data.

## 🚀 Key Features
* **Live Weather Data:** Instant updates fetched via **OpenWeatherMap API** using the Axios library.
* **Dynamic Visual Experience:** The UI background and icons change dynamically (e.g., Clear, Clouds, Rain, Snow) using high-quality .gif visuals to match current conditions.
* **Global Search:** Efficiently search and retrieve weather metrics for any city worldwide.
* **Comprehensive Metrics:** Displays real-time temperature, humidity, wind speed, and "feels like" indicators.

## 🛠️ Tech Stack
* **Frontend:** **React.js** (utilizing Functional Components, Hooks like `useState` and `useEffect`).
* **API Handling:** **Axios** for robust asynchronous HTTP requests.
* **Styling:** CSS3 with dynamic state management for theme transitions.
* **Tools:** Git for version control, Visual Studio Code.
